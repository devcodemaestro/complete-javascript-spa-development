미션2. 간단한 연산자 선택기

두 숫자와 연산자 기호 중 하나("+", "-", "\*", "/")를 전달하면, 해당 연산의 결과를 출력하는 함수를 switch문을 사용해 만들어보세요.

```js
//예시 실행
calculate(10, 5, "+"); // 15
calculate(10, 5, "*"); // 50
```
